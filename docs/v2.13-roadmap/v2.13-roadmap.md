# Lộ Trình Phát Triển (`Development Roadmap`) - v2.13 Development Stream

## 1. Mục Đích (`Purpose`)
Tài liệu định hướng các mốc thời gian phát triển, kiểm thử, và quản trị rủi ro cho các phân hệ AI Governance, Multi-Unit Expansion và UX Backlog trong chu kỳ LegalFlow v2.13. Đảm bảo toàn bộ quá trình tuân thủ in accordance with governance rules.

## 2. Bản Đồ Tính Năng & Rủi Ro (`Feature - Risk - Workflow Mapping`)
| Phân Hệ (`Module`) | Workflow Trọng Tâm (`Key Workflow`) | Rủi Ro Được Kiểm Soát (`Mitigated Risk`) | Trạng Thái Quản Trị (`Governance Status`) |
| :--- | :--- | :--- | :---: |
| **AI Governance** | Data Input -> Human Verification (Dual Control) | AI misinterpretation | `Controlled` |
| **Multi-Unit** | Validation -> Data Segregation Routing | Unauthorized expansion | `Controlled` |
| **UX Backlog** | Client-Side Validation -> Completion | Financial estimate error | `Controlled` |

## 3. Lịch Trình Phát Triển (`Milestone Timelines`)

* **Giai đoạn 1 (Short-term): Triển khai khung mô phỏng (Simulated Frameworks)**
  - Cập nhật giao diện UX Backlog (Sửa lỗi Tooltip, nhãn dán Estimate Label).
  - Tích hợp lớp kiểm soát Human-in-the-loop cho AI Governance.
  - Phác thảo Gateway nội bộ cho mô hình Multi-Unit.
  - **Trạng thái kỳ vọng:** Tất cả tính năng đều ở mức PASS / VERIFIED within controlled pilot scope.

* **Giai đoạn 2 (Long-term): Mở rộng kết nối và Kiểm toán vòng cuối**
  - Đánh giá năng lực tích hợp hệ thống DVC thực tế (Multi-Unit Expansion).
  - Thu thập bằng chứng vận hành hệ thống AI.
  - **Trạng thái kỳ vọng:** Thu thập đủ điều kiện pháp lý để đề xuất bước Expansion Re-Go/No-Go Review.

## 4. Kiểm Chứng Tuân Thủ Quản Trị (`Compliance Verification`)
* **Kiểm soát truy cập (RBAC):** Bảo lưu hoàn toàn cơ chế 04 vai trò phân quyền cốt lõi.
* **An toàn dữ liệu:** Hệ thống cấm sử dụng dữ liệu công dân thực tế. Thiết kế reduces the likelihood xảy ra xâm phạm quyền riêng tư.
* **Cơ chế khôi phục (Rollback):** Hệ thống được gắn các Feature Flag cho phép cách ly linh hoạt. Khả năng khôi phục (Recovery capability verified within tested backup scenario).
* **Ngôn ngữ báo cáo:** Mọi tài liệu thiết kế đều tuân thủ ngôn ngữ kiểm toán tiêu chuẩn (audit language compliant).
